Простой консольный калькулятор на Python

Учебный проект для выполнения базовых арифметических операций с валидацией ввода и защитой от ошибок.

 Функционал
* Сложение, вычитание, умножение и деление.
* Защита от падения при делении на ноль.
* Обработка некорректного ввода (защита от ввода букв вместо чисел).
* Красивое форматирование результата (автоматическое приведение к целому числу при делении без остатка).

 Требования
* Python версии 3.6 или выше.

 Запуск проекта
1. Клонируйте репозиторий:
   bash
   git clone github.com
   
2. Перейдите в папку с проектом:
   bash
   cd caluclator
   
3. Запустите скрипт:
   bash
   python main.py
   

 Пример работы
Первое число:10.0
Второе число:2.0 
1. +
2. -
3. *
4. /
